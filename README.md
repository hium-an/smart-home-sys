# Group One - Smart Home System

Đồ án môn Lập Trình Nhúng

## Thành viên nhóm

| MSSV | Họ và tên |
|------|-----------|
| 25TX610004 | Nguyễn Chí Khang |
| 25TX810003 | Trương Văn Giáp |

## Mô tả dự án

Hệ thống nhà thông minh (Smart Home System) được lập trình trên vi điều khiển, bao gồm các chức năng đọc cảm biến và điều khiển thiết bị trong nhà.

## Công cụ sử dụng

- **Keil uVision** — IDE và trình biên dịch ARM
- **Proteus** — mô phỏng mạch điện

## Cấu trúc thư mục

```
src/      Source code C
inc/      Header files
asm/      Assembly (delay, ...)
keil/     Keil uVision project files
sim/      Proteus simulation
docs/     Báo cáo đồ án
build/    Output biên dịch (git-ignored)
```
